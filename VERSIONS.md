# Kool AI Landing Page Versions

## Overview

This project contains multiple design iterations of the Kool AI landing page. Each version is a self-contained Next.js application.

---

## v.01

**First iteration** of the Kool AI landing page.

- Dark theme with cyan accent colors
- Sections: Hero, Problems, Solution Demo, Features, Integrations, Testimonials, CTA, Footer
- Built with Next.js 16, React 19, Tailwind CSS 4, Framer Motion
